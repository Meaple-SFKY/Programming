热门故事

足不出户的春节，苏梅岛山顶别墅迷失记

几个小时前，大家还在“钢筋水泥森林”里，乘坐曼航抵达苏梅岛，以最放松的心情入住苏梅岛山顶别墅，愉快地布置春节事宜。
我们春节的关键词是“足不出户”。别墅面朝东方，是欣赏椰林日出的好地方，6:00整，睡眼惺忪地走到泳池边，深深吸一口气，看着椰林的颜色随着日出一点点变成金色，来，喝一杯热牛奶，开启美好的一天。
厨房被男人霸占了，新鲜果汁和凉面后，大家满怀欣喜地各自拉开欢乐假期的篇章。有在屋里练毛笔字的，有在泳池边小憩的，有一直泡水里扑腾的，有拿着相机一直记录的，有热爱厨艺研究除夕晚餐的……
足不出户，却精彩丰富。旅行的态度是什么？ 是“情调”，情趣和格调。情趣是方向，是横坐标。“有趣”有几乎无限的方向。用异国食材做料理，在别人的泳池边做自己的日常运动……都能撬动一点点心情。格调是量级，是纵坐标。用手机，无人机，单反……是量级的差别。旅行中无数的变量＋组合，把心情调节得有温度有起伏，这件事如此重要。
