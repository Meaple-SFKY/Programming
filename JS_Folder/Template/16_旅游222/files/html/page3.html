热门故事

在冲绳听见涛声，然后迷失在绿色里

在冲绳有名的美丽海水族馆附近，有一处相对鲜为人知的景点，名为“备瀨福木林道”，这是一片非常美丽的福木林，紧邻海岸，夏天到访时，是艳阳下难得的清幽避暑地，但要是过于悠闲，一个不小心，说不定就会迷失其中。
福木在以前就时常被冲绳人民用作对付台风的防风林，这片备瀨地区的防风林就是因为种植了大量的福木，而沿着海岸形成了一片幽静的林荫道，如今的林道旁还住着一些人家，专门负责维护这片树林。这里的福木大多都有着超过300年以上的树龄，所以弥漫着一种古老而神秘的气息，踏入其中，就像踏入了异世界。
我们就是在里头走着走着就失去了方向感，明明能听见附近大海的涛声，但就是找不到出去的路。好在遇上了维护树林的一位老奶奶，用蹩脚的日语询问了出口，才得以“逃离异界”，也算是一次小小的冒险呢~（才怪）
