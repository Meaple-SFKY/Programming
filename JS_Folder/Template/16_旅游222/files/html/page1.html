热门故事

在金程武骑车的地方吃个便当

两旁稻穗随风摇曳，一条笔直的乡间小路延伸至远处的山间…因Mr. Brown Coffee的广告而蹿红，伯朗大道就这样将一大片稻田一分二。但Mr. Brown再出名也难敌男神，因为金城武为长荣航空拍摄的另一辑名为《I SEE YOU》的广告，伯朗大道硬生生地被世人改名为金城武大道。
因为年轻人都不愿意呆在老家种地，所以池上乡的居民大多都是上了年纪的，人口只有五千。要说哪里人口密度最高，大概是金城武在广告中奉茶的那颗大树。只是比起在树下无聊打卡，从大道的这边疾驰至另一头似乎更为畅快。
即使火车每次只会在这停站1分钟，但是乘客总是会在停站期间下车购买著名的池上便当。皆因让池上更早驰名的，并不是伯朗咖啡，也不是金城武，而是他们随处可见的稻米。不过最好吃的池上便当，并不是提篮在车站内售卖的那一家，而是出车站后第一个十字路口的这一家，门面内同时贩卖着琳琅满目的当地手信。所以不如花上一个下午在池上放空，啖啖正宗的池上便当。
