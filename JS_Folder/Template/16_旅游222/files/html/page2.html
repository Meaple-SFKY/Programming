热门故事

大哥的绿岛监狱

遇上那一年最厉害的台风，在太平洋上颠簸了一个小时，也搞不清楚究竟是风雨拍打着窗户，还是海浪拍打着窗户，庆幸我们踏上绿岛的时候还没吐。
租两辆机车环岛，不开手机导航也无需担心迷路，因为风景都沿着海岸线按顺序播放，只是记得给机车充好电。
机车的速度不快，突突突地绕过了牛头山，绿岛山庄就在附近。绿岛山庄，旧名绿岛监狱，是白色恐怖时代人权斗士们的监禁之所，如今则成了绿岛最为知名的旅游符号。绿岛山庄保留了当年的建筑，规划建立了人权纪念园，纪念在戒严期间为民主思想奋斗的人士。
绕到山的那边，往朝日温泉的路上遇上了无数要过马路的小螃蟹。原来路边“陆蟹出没”的告示牌并不是骗人的把戏，只是…驻车等待了一只又一只后，确实无法再等待更久了，于是迫不及待地继续追逐朝日温泉。能称之朝日的，自然是看到日出的温泉，而且是罕有的在海边礁石间形成的海水温泉，世界上只有三处，另外两处在日本和意大利。幸运的话，确实能泡着咸水温泉看日出。
如果你从花莲走到台东，便直接往南奔垦丁，也许你会懊悔自己就此错过了这颗沧海遗珠。
