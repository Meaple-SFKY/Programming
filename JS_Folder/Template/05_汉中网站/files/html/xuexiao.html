  • 勉县一中

    勉县武侯中学，曾叫“勉县一中”，位于陕西省汉中市勉县，坐落于定军山下，汉水河畔，汉中市勉县武侯祠西，古阳平关诸葛亮读书台东。学校创建于1938年，曾名武侯中学，陕西省首批重点中学、标准化高中，后更名为“陕西省勉县第一中学”，于2016年恢复曾经的校名——“勉县武侯中学”。 学校素以办学理念先进、办学思想端正、管理严格、校风优良、学风浓厚、教学质量高而闻名于省内外，被社会各界誉为"普教明珠，人才摇篮"，成为莘莘学子向往之地。

  • 宁强县天津高级中学

    宁强县天津高级中学是一所省级标准化高中，其前身是创建于1942年的宁强县第一中学，1958年始设高中班，1992年转为完全高中。2008年5·12汶川地震波及宁强，学校遭受严重损毁，被列为灾后重建重点项目，迁址县城七里坝新区，由天津市对口援建。2008年8月破土动工，翌年9月落成并投入使用，并将学校更名为宁强县天津高级中学。
